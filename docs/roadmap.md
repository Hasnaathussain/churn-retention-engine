# Roadmap

## Highest Priority

- Replace preview/in-memory persistence with Postgres and Redis
- Add migrations and repositories
- Implement real email delivery
- Implement real OAuth
- Add queue-backed prediction refresh and reporting jobs
- Add durable audit logging

## Next Product Milestones

- Full billing plan enforcement
- Custom report builder
- Slack alerting and bot workflows
- CSV import pipeline
- Multi-organization switcher backed by database state
- Richer admin console and observability

## Known Limitations

- Demo data is still a core fallback path
- The product API is only partially production-hardened
- Billing and auth are not yet complete enough for a paid public launch without more work
